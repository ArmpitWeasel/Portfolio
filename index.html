BTS SIO SISR - E5 Professional Project

NAS Storage and Automated Backup Service

Download report PDF

During my professional placement, I deployed a centralized storage service to improve file organization, backup reliability, and secure remote access for the company owners. The solution reused existing hardware and added RAID 1 storage, Samba shares, automated backups, and Tailscale access.

Context Company placement
Period 10 Jan - 31 Jan 2026
Service delivered NAS, backups, remote access
Main technologies Ubuntu Server, Samba, RAID 1, Bacula, crontab, Tailscale

Objectives

  • Centralize shared company files on a reliable internal service.
  • Reuse an existing PC to control the project budget.
  • Protect data with RAID 1, local USB backup, and off-site cloud backup.
  • Provide secure remote access without exposing the NAS directly to the Internet.

Result

  • Operational Samba file shares with user access rights.
  • Automated backup jobs using Bacula and crontab.
  • Local and remote backup strategy inspired by the 3-2-1 rule.
  • Remote access validated through Tailscale for authorized owners.

Acquired E5 Competencies

01
Manage IT assets

Inventory of reusable hardware, selection of new disks, budget control, and integration of the NAS into the existing IT environment.

02
Work in project mode

Project broken into phases: needs analysis, inventory, budget, installation, configuration, backups, testing, and production handover.

03
Provide an IT service to users

Delivery of a usable storage service with Samba shares, access rights, automated backups, restoration checks, and secure remote access.

Service NAS de stockage et de sauvegardes automatisées

Télécharger le compte rendu PDF

Lors de mon stage, j'ai mis en place un service de stockage centralisé afin d'améliorer l'organisation des fichiers, la fiabilité des sauvegardes et l'accès distant sécurisé pour les propriétaires. La solution réutilise du matériel existant et ajoute du RAID 1, des partages Samba, des sauvegardes automatisées et un accès Tailscale.

Contexte Stage en entreprise
Période 10 janv. - 31 janv. 2026
Service rendu NAS, sauvegardes, accès distant
Technologies principales Ubuntu Server, Samba, RAID 1, Bacula, crontab, Tailscale

Objectifs

  • Centraliser les fichiers partagés de l'entreprise sur un service interne fiable.
  • Réutiliser un ancien PC afin de maîtriser le budget du projet.
  • Protéger les données avec du RAID 1, une sauvegarde USB locale et une sauvegarde cloud hors site.
  • Donner un accès distant sécurisé sans exposer directement le NAS sur Internet.

Résultat

  • Partages de fichiers Samba opérationnels avec droits utilisateurs.
  • Tâches de sauvegarde automatisées avec Bacula et crontab.
  • Stratégie de sauvegarde locale et distante inspirée de la règle 3-2-1.
  • Accès distant validé via Tailscale pour les propriétaires autorisés.
